## Project Overview

Design and implementation of a **12V DC motor speed controller** using a **555 timer-based chopper circuit** with PWM control. The project achieves precise speed regulation with high efficiency for variable load conditions.

**Key Features:**

* Continuous speed range: **360–2830 RPM**
* Adjustable duty cycle: **12–88%** via potentiometer
* Motor protection using **flyback diode**
* Overall efficiency: **78.5%**
* Cost-effective and practical for educational/light industrial use

---

## Circuit & Implementation

* **PWM Generation:** NE555 in astable mode
* **Switching:** TIP122 Darlington transistor
* **Motor:** 12V, 24W DC motor
* **Simulation & Testing:** MATLAB/Simulink + hardware prototype
