# DC Motor Speed Control Using Chopper Circuit

## Description
This project demonstrates the **speed control of a DC motor** using a **chopper circuit**. The chopper adjusts the voltage supplied to the motor, enabling precise speed regulation and real-time response. The project highlights the relationship between duty cycle and motor speed.

## Features
- Variable speed control using a chopper circuit.
- Real-time motor response to voltage changes.
- Demonstrates integration of electronics and control theory.

## Components / Technologies
- DC Motor
- Chopper Circuit (IGBT/MOSFET or relevant switching devices)
- Microcontroller / Arduino (if applicable)
- Circuit Simulation Tools (Proteus, MATLAB, or similar)

## How to Run / Simulate
1. Set up the DC motor and chopper circuit according to the schematic.  
2. Apply input voltage and adjust duty cycle to control motor speed.  
3. Observe motor response and measure speed at different duty cycles.
